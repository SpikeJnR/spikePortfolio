Посмотреть проекты

01.Обо мне

Мой путь в веб-разработке начался в 2024 году, когда я впервые попробовал собрать собственный сайт — с тех пор это стало не просто хобби, а настоящей страстью.

Я стремлюсь создавать удобные, доступные и визуально привлекательные приложения, которые приносят реальную пользу пользователям. В процессе разработки для меня важны внимание к деталям, чистота кода и постоянный рост.

За пределами редактора кода я жгу виртуальные покрышки на автосимуляторе, погружаюсь в мир «Звездных войн», наматываю километры на шоссейнике или задаю темп на беговой дорожке. Динамика, скорость и движение — мой источник вдохновения. Эти увлечения заряжают энергией, учат быстроте принятия решений, стратегии, выносливости и воображению — качествам, ценным для работы.

Вот несколько технологий, с которыми я работаю:

- JavaScript (ES6+)
- React

- HTML5/CSS3

02.Учебные проекты

Six-cities

«Шесть городов» — сервис для путешественников, не желающих переплачивать за аренду жилья. Выбирайте один из шести популярных городов для путешествий и получайте актуальный список предложений по аренде. Подробная информация о жилье, показ объекта на карте, а также лаконичный интерфейс сервиса помогут быстро выбрать оптимальное предложение.

React Typescript JavaScript HTML5/CSS3

Big trip

«Большое путешествие» — современный сервис для настоящих путешественников. Сервис помогает детально спланировать маршрут поездки, рассчитать стоимость путешествия и получить информацию о достопримечательностях. Минималистичный интерфейс не даст повода отвлечься и сфокусирует внимание на планировании путешествия.

JavaScript HTML5/CSS3

Kekstagram

«Кекстаграм» — сервис просмотра изображений. Пользователям предоставлена возможность загружать свои фотографии или просматривать фотографии, загруженные ранее другими пользователями.

JavaScript HTML5/CSS3

Sedona

«Седона» — проект о небольшом городке в Аризоне для поиска гостинницы.

HTML5/CSS3
